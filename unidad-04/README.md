## Descripción

Esta entrega aplica el método **Attribute-Driven Design (ADD)** para diseñar la arquitectura del sistema **SmartRoad** — una plataforma de gestión de infraestructura vial inteligente. El proceso ADD parte de los atributos de calidad priorizados, los drivers arquitectónicos y las restricciones del sistema para derivar decisiones de diseño justificadas.

El diagrama de arquitectura fue elaborado en **Draw.io** como complemento al modelo C4 previamente desarrollado en la Unidad 3.
